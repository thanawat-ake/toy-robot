# Toy Robot (Console)

A simple console simulation of a toy robot moving on a 5x5 table.

## Requirements
- Java 21+
- No global Maven installation required (Maven Wrapper included)

## Build & Run
1. Run Tests  
   ./mvnw test


2. Package the application  
   ./mvnw package


3. Run the application  
   cat sample.txt | java -jar target/toy-robot-1.0.0.jar
